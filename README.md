# Аналитика данных и машинное обучение: сборник кейсов

Этот репозиторий содержит решения четырёх практических кейсов в области анализа данных и машинного обучения.

## Кейс 1: Анализ потребительского поведения для оптимизации бизнес-стратегии ритейлера

**Описание ситуации:** Крупная розничная сеть супермаркетов столкнулась с проблемой оптимизации ассортимента и маркетинговых кампаний. У компании есть большой массив данных о покупках клиентов с дисконтными картами, но эти данные используются недостаточно эффективно.

**Контекст ситуации:** Компания работает на высококонкурентном рынке ритейла, где маржинальность постоянно снижается, а конкуренция растет.

**Задания:**
1. Провести разведочный анализ данных (EDA)
2. Выполнить сегментацию клиентов
3. Выявить ассоциативные правила
4. Создать информативные визуализации
5. Разработать рекомендации по оптимизации

**Датасет:** Online Retail Dataset с Kaggle (541 909 транзакций)

## Кейс 2: Разработка системы рекомендаций для стримингового сервиса

**Описание ситуации:** Стартап в области онлайн-стриминга аудиоконтента хочет улучшить взаимодействие с пользователями путем персонализации рекомендаций.

**Задания:**
1. Анализ данных о пользователях и их взаимодействии с контентом
2. Разработка моделей рекомендаций
3. Разработка метрик для оценки качества
4. Реализация механизма решения проблемы "холодного старта"

**Датасеты:** Last.fm Dataset, MovieLens 25M Dataset

## Кейс 3: Анализ тональности и выявление трендов в отзывах о продукте

**Описание ситуации:** Производитель бытовой техники выпустил новую линейку "умных" устройств и собрал значительное количество отзывов пользователей.

**Задания:**
1. Разработка системы сбора и предобработки текстовых отзывов
2. Анализ тональности (sentiment analysis)
3. Тематическое моделирование
4. Анализ трендов во времени
5. Выявление критичных проблем и возможностей для улучшения

**Датасет:** Amazon Product Reviews (3.8 млн обзоров)

## Кейс 4: Оптимизация логистической сети для снижения углеродного следа

**Описание ситуации:** Международная логистическая компания объявила о стратегии по снижению своего углеродного следа на 50% к 2030 году.

**Задания:**
1. Анализ текущей логистической сети и расчет углеродного следа
2. Разработка модели для расчета выбросов CO2
3. Создание оптимизационной модели
4. Разработка геопространственной визуализации
5. Предложение плана внедрения оптимизированной стратегии

**Датасет:** Global Supply Chain Dataset

## Технологии

- Matplotlib, Seaborn, Plotly

- Геоанализ: Folium, GeoPandas
